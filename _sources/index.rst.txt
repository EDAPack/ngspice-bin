ngspice-bin Documentation
=========================

**ngspice-bin** is a portable, pre-built distribution of
`NGSpice <https://ngspice.sourceforge.io/>`_ for Linux.  It provides a
self-contained binary tarball that can be dropped into any project without
requiring a system-level NGSpice installation.
